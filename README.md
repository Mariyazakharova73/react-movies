**Описание проекта**

Проект "react-movies". Поиск фильмов. Информация о фильмах загружаается с сервера. Реализован фильтр по фильмам и сериалам.

**Технологии и инструменты**

* HTML5
* CSS3
* Java Script
* React
* https://www.omdbapi.com/
* https://materializecss.com/
